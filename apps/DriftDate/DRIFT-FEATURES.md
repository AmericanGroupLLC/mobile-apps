# Drift — FEATURE INVENTORY (v1.0)

## In v1

### Discovery

- Layer selector: **Server · State · County · ZIP**, switchable from the home tab.
- Layer chips on every profile card: "same ZIP", "same county", "100 mi away".
- Per-layer pool sizes shown above the feed: "8 nearby on ZIP, 42 in your county, 120 in California".

### Profile

- 6 photos, one **must** be tagged the verification selfie.
- Voice prompt (30s `AVAudioRecorder` / `MediaRecorder` clip).
- Intent badge — Dating / Serious / Friendship / Open.
- Vibe tags — small fixed taxonomy (~30 entries).
- Three Hinge-style structured prompts.

### Matching loop

- **Wave** or **Pass** in proximity layers (tap, no swipe).
- **Like-a-prompt** in server mode.
- Mutual Wave / Like → chat unlocks (verification-gated).
- No "super likes" paywall.

### Chat

- Realtime via Supabase Realtime (Postgres LISTEN/NOTIFY).
- **Three reply suggestions on every screen entry** — Casual / Context /
  Playful — generated by an Edge Function (LLM provider, env-keyed).
- Tone evolves: `.slow` → light prompts; `.energetic` → playful escalation;
  `.deep` → thoughtful follow-up; `.meetupReady` → public-meetup
  suggestions. (`Conversation.tone` column updated by `ToneClassifier`.)

### Safety

- Selfie verification before messaging unlocks (AWS Rekognition CompareFaces).
- Report / block / mute on every profile + chat.
- Anonymous display name until mutual Wave + verification.
- Location fuzzing: ZIP-prefix-3 / county-FIPS / state-code only.
- Public-meetup framing — no private location share.
- Screenshot disclosure on iOS + Android.

### Watch surfaces (Apple Watch + Wear OS)

- Match notification complication — *"Sara matched"* + quick reply.
- Glanceable layer indicator (server / state / county / zip) + unread match count.
- Wave-back tile — 1-tap accept on a pending Wave.

### Settings

- Layer toggles · intent · invisibility · pause discoverability · 12/24-hour
  · theme (system/light/dark) · Sentry/PostHog opt-ins · erase-all-data ·
  account deletion.

## Explicitly NOT in v1

- Bluetooth-LE / WiFi-Direct discovery → v1.1.
- Venue mode → v1.1.
- IAP / RevenueCat → v1.1.
- Voice / video calling → v2.
- Stories / live presence beyond match notifications → v2.
- Cross-platform photo / message search → v2.
- AI auto-classification of profiles → v2.
- Group / friend invites → v2.
