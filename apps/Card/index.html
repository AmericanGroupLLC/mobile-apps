Four quick-capture surfaces

📱 In-app composer

Open Card. The text field already has focus. Type, hit return, it's saved. No screen between you and the field.

📤 iOS Share Extension

Select text in any app → Share → Card – Save. The text becomes a Card without ever launching Card itself.

⌚ Watch complication

Add the "Quick capture" complication to any watch face. Tap → dictate → done. Rolls into the same feed on your phone.

⚡ Android Quick Settings tile

Drag the Card tile into Quick Settings. Two-finger swipe → tap → voice composer is up. No home-screen widget required.

Domain logic lives in a single shared core (CardCore Swift Package on Apple, :core Kotlin/JVM module on Android). Tests are mirrored case-for-case across both stacks.

Private by design

No account. No login.

Card never asks who you are. There is no server it could ask.

Local storage only

Every Card stays on the device it was written on. No cloud sync, no backup beacons, no telemetry by default.

Crash + analytics opt-in

Sentry and PostHog wrappers are off by default. Toggle in Settings if you want to help.

Zero camera, zero location

The only sensitive permission is microphone, requested only when you tap the watch's voice composer.
